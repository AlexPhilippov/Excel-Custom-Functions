You can use this sample to try Custom Functions. Note that this file is only for the instructions when sideloading the add-in on Windows.

Note: If this is your first time sideloading this add-in, or if the functions don't appear in the Excel auto-completion list, re-insert the add-in using "Insert" > "My Add-ins".

Then, try something like the following:

  • =CONTOSO.ADD(3, 4)
  • =CONTOSO.ADD(A1, A2)
  • =CONTOSO.ADDASYNC(100, 200)
  • =CONTOSO.INCREMENT(5)

For more info see https://aka.ms/customfunctions.
