You can use this sample to try Custom Functions. Note that this file is only for the instructions; the file used by the runner of the Custom Functions is inside of the "customfunctions/index.html" folder.

Try something like:

  • =CONTOSO.ADD(3, 4)
  • =CONTOSO.ADD(A1, A2)
  • =CONTOSO.ADDASYNC(100, 200)
  • =CONTOSO.INCREMENT(5)

For more info see https://aka.ms/customfunctions.
